## 介绍
- 就是一个很普通的色图插件呀！
- 加了一点功能，但是保留了生草的代码设计，我故意的，这样你才知道我是nooooob。
- 启动后群聊或私聊setu -h看菜单。
- 一切后果自负嗷，别在不允许用的地方用。
